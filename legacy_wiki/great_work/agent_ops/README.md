# Great Work Agent Operations

This folder contains operating material for agents working on the `great_work` wiki. It is not part of the public-facing wiki index.

- [`PLAN.md`](PLAN.md) - source-first expansion plan, lead queue, source standards, and research operating system.
- [`EDITORIAL_PLAN.md`](EDITORIAL_PLAN.md) - plan for editorial agents who audit, rerank, refresh, and maintain existing entries.
- [`prompts/`](prompts/) - reusable prompts for different agent roles.
- [`editorial_passes/`](editorial_passes/) - iterative pass ledgers for whole-wiki audits.
- [`_messy_thoughts.md`](_messy_thoughts.md) - scratchpad for raw leads and unresolved fragments.
- [`sams-notes.md`](sams-notes.md) - subjective notes and tier instincts to preserve as context.

Public wiki readers should generally start at [`../README.md`](../README.md).
