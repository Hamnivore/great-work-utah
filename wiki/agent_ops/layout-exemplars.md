## How to use this file

When you write or improve a real wiki entry that uses a layout *well* — meaning the layout matches the content's natural shape, the imagery is real and cleared, and the page looks like something the team would be proud to show — append a line under the matching heading:

```markdown
- `wiki/work/some-real-entry.md` — why this is a good example (~10 words)
```

A "strong" exemplar means: a reader landing on the page would understand what good looks like for this layout, and would learn the convention by example. One paragraph of justification, max.
